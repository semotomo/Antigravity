# アカウント作成機能と設定オプション画面の実装計画

本計画は、マスター管理者が「わんわん」用のアカウントを容易に新規作成できる場所を提供し、さらに各店舗のPOSグループ連携設定（店舗マッピングID等）をWeb画面から変更できるようにするためのものです。

## 変更内容の概要

1. **設定オプション用 Server Action の実装**
   - パス: `next_app/lib/actions/options.ts`
   - **アカウント作成**: 現在ログイン中のマスター管理者のセッションを壊さないように、`persistSession: false` で Supabase クライアントを作成し、指定されたメールアドレス、パスワード、および `store_type: 'wanwan' | 'master'` を指定してサインアップを処理します。
   - **店舗設定更新**: `stores` テーブルの各店舗に対する `pos_group_id` および `pos_group_name` を更新・保存します。

2. **設定オプション UI コンポーネントの作成**
   - パス: `next_app/components/options/OptionsBoard.tsx`
   - **アカウント追加フォーム**: メールアドレス・パスワード・アカウント権限（わんわん店舗 / マスター管理者）を入力してアカウントを新規作成できるUI。
   - **店舗マッピング編集**: `stores` に登録されている店舗一覧を取得し、それぞれの `pos_group_id` / `pos_group_name` を変更・保存するUI。

3. **設定オプションメインページの作成**
   - パス: `next_app/app/(dashboard)/options/page.tsx`
   - 管理者権限を判定し、マスター管理者以外がアクセスした場合は不正アクセスエラー（403）または売上一覧へのリダイレクトを行います。

4. **サイドバーメニューへの「設定オプション」統合**
   - パス: `next_app/components/layout/SideNav.tsx`
   - ログインユーザーが `storeType === 'master'` の場合のみ、サイドバーメニューに「設定オプション」（歯車アイコン）を表示します。
