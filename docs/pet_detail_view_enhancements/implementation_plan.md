# 性別色分け・ワクチン追加・値下げ価格の赤色化の実装計画

詳細リスト表示および詳細モーダルにおいて、性別の視覚的な色分け、ワクチン情報の配置、値下げ前価格の打消し線、値下げ後価格のみの赤色強調表示を行い、管理画面としての使いやすさをさらに向上させます。

## ユーザーレビューが必要な事項
なし（フロントエンドのReact/TypeScriptの改修のみで完結します）。

- **性別の色分け表示**:
  - 女の子はピンク（`text-pink-600`）、男の子はブルー（`text-sky-600`）になるようクラスを出し分けるヘルパーロジックを実装します。
