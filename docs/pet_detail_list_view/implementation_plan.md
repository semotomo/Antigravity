# 詳細リスト表示切り替え機能の実装計画

生体一覧画面において、最初から写真を全て読み込んで重くなるのを防ぎつつ、必要な時だけ「表示中の生体（絞り込み結果）」を写真・価格付きの横一列（リスト形式）で詳細表示できる機能を追加します。

## ユーザーレビューが必要な事項
なし（すべてフロントエンドのReact/TypeScriptの改修のみで完結します。DBへの変更はありません）。

---

## 提案する変更内容

### 1. フロントエンド（一覧画面）の改修
#### [MODIFY] [PetsBoard.tsx](file:///C:/Users/kirik/Desktop/Antigravity/next_app/components/pets/PetsBoard.tsx)
- **状態管理の追加**:
  - 詳細リスト表示のオン/オフを管理する state `showDetailList` を追加します。
- **切り替えUIの追加**:
  - 上部フィルターエリアに、「詳細（写真付き）表示」のチェックボックスを追加します。
- **レイアウトの動的切り替え**:
  - `showDetailList` が `false`（デフォルト）の場合：
    - 従来のグリッド形式（画像なし、文字のみ、5列）で軽快に表示します。
  - `showDetailList` が `true` の場合：
    - 横一列のリスト（行）形式（縦に並ぶレイアウト）に切り替えます。
    - 各行に「小さめの角丸写真（画像が無い場合は犬猫アイコン）」「品種・管理番号」「毛色・性別」「誕生日・所属店舗」「生体価格（price_text優先）」を表示します。
    - 行全体をクリック可能にし、クリック時に詳細モーダルが開く仕様を維持します。

### 2. 手動検証

- 写真が小さめのサイズ（80px四方程度）で綺麗に収まり、デザインが崩れていないことを確認します。
- 検索キーワードや店舗で絞り込んだ際、表示されている生体だけが正しくリスト表示されることを確認します。
