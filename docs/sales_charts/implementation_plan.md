# 実装計画書 (Implementation Plan) - ABC分析グラフ機能の実装

本計画書は、ABC分析画面（`/sales/abc`）において、売上貢献度の高い商品や各ランクの売上比率を視覚的かつ直感的に把握できるようにするための「モダンでプレミアムなビジュアルダッシュボード（グラフ）」の実装計画です。

---

## 🎯 実装の目的と成功基準
- **目的**: 現在テーブル一覧（DataTable）でのみ表示されているABC分析結果を、リッチなビジュアルグラフで表現し、どの商品が売上を牽引しているか、および各ランクの貢献度をひと目で把握できるようにします。
- **成功基準**:
  - ABC分析画面の期間検索（From/To）、店舗フィルター、カテゴリフィルターと**完全にリアルタイム連動**する。
  - レスポンシブ対応であり、モバイルでもPCでも崩れない。
  - Recharts を駆使し、滑らかなホバーエフェクトや美しいグラデーションを持つプレミアムなデザインを実現する。

---

## 📦 追加するパッケージ
React 19 および Next.js 16 (Turbopack) の環境下で最も美しく動作し、リッチなアニメーションやカスタムツールチップが実装可能な **`recharts`** ライブラリを導入します。

- パッケージ名: `recharts`

## 🎨 グラフのUI/UXデザイン設計
ABC分析の真価を引き出す、プレミアムなダッシュボードデザインを採用します。

1.  **左側: 「売上貢献度 TOP15 商品」の横棒グラフ (Horizontal Bar Chart)**
    - 商品名がはっきりと読めるよう、横型の棒グラフを採用します。
    - 売上金額が最も高い上位15商品を表示します。
    - 各バーの色は、商品の属するランク（A: グリーン, B: ブルー, C: オレンジ）に応じて美しく色分けされます。
2.  **右側: 「ランク別売上シェア」のドーナツチャート (Donut Pie Chart)**
    - A・B・Cランクが全体の総売上高に占める比率を視覚化します（Aランクが大部分を占めるABC分析の特性が綺麗に現れます）。
    - 中央に「総売上金額」を表示するリッチなドーナツスタイルを採用します。
3.  **シームレスな連動**
    - 「期間」「店舗」「カテゴリ」フィルターを適用すると、サーバーから新しいデータが取得され、それに連動してグラフが滑らかにアニメーション変化します。
