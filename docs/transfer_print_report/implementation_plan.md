# 実装計画書：店舗間移動・物品使用の月次レポート印刷機能の追加と不要バナーの削除

本ドキュメントは、店舗間移動・物品使用画面における不要な案内バナーの削除、および月次レポートをスッキリした印刷用レイアウトで出力する機能の実装計画です。

## 目的と背景

1. **画面の整理（バナー削除）**
   - 登録後の確定に関する不要な案内テキストバナーを削除し、画面をよりスッキリと整理します。
2. **月次レポートの印刷対応**
   - これまで画面上での確認のみだった店舗間移動・物品使用の履歴について、月ごとに集計し、移動ルート（または物品使用）ごとに見やすく整理された枠線付きのレイアウトで印刷（紙・PDF）できるようにします。

## ユーザーレビューが必要な点

- 印刷レイアウトは、装飾的なヘッダー線を廃止し、タイトルと対象月を1行にまとめることでコンパクト化しました。
- 各グループの小計は出力せず、月全体の総合計金額のみを出力するシンプルな構成にしています。
