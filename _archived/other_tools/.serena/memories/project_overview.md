# プロジェクト概要 — シフト作成ツール

## 概要
Streamlit製のシフト自動作成ツール。看護・介護・飲食等の現場向けに、スタッフの勤務シフトを自動生成する。

## 技術スタック
- **フレームワーク**: Streamlit
- **言語**: Python 3.14
- **主要ライブラリ**: pandas, numpy
- **実行方法**: `streamlit run app.py`

## 主な機能
- スタッフ情報の入力・管理（名前、役割、勤務可能区分）
- 🎭 動的役割カスタマイズ（追加/削除/名前変更/必要人数/優先順位）
- スタッフ別優先役割設定
- 休暇希望・出勤指定の入力（コンフリクト警告付き）
- 制約条件の設定（最低人数、最大連勤数等）
- 曜日別の目標人数設定
- シフトの自動生成（ビームサーチアルゴリズム）
- 📊 公平性ダッシュボード（土日出勤・役割分布・勤務バランス）
- 🔧 手動微調整・📱 プレビューモード
- 📈 シフト履歴管理（自動保存・閲覧・削除）
- 🏪 店舗管理（複数店舗の切替・追加・削除・名前変更）
- 📝 メモ・連絡事項欄
- 結果のCSVエクスポート
- 設定の保存・読込（JSON形式）
