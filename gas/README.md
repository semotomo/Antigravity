# 売上CSV自動転記スクリプト — セットアップ手順

## 概要
Googleドライブに保存されたPOS売上CSVを、スプレッドシートの「貼り付け用」シートに自動転記するスクリプトです。

---

## セットアップ手順

### STEP 1: CSVの保存先フォルダを準備
1. Googleドライブに **CSV保存用のフォルダ** を作成する（例: `POS売上CSV`）
2. フォルダを開いた状態で、ブラウザのURLから **フォルダID** をコピーする

```
https://drive.google.com/drive/folders/1ABC_xyzDefGHIjklMNO
                                       ^^^^^^^^^^^^^^^^^^^^^^^^
                                       ↑ この部分がフォルダID
```

### STEP 2: GASスクリプトを登録
1. 対象のGoogleスプレッドシート（月計AI用）を開く
2. メニューから **「拡張機能 → Apps Script」** をクリック
3. エディタに `importCSV.gs` の内容を **全文コピペ** する
4. コード上部の `CONFIG` セクションを編集:

```javascript
const CONFIG = {
  CSV_FOLDER_ID: '1ABC_xyzDefGHIjklMNO',  // ← STEP 1でコピーしたIDに変更
  CSV_FILE_PATTERN: '月計用',               // ← CSVファイル名に含まれるキーワード
  TARGET_SHEET_NAME: '貼り付け用',           // ← 転記先シート名
  CSV_ENCODING: 'Shift_JIS',                    // ← 文字化けする場合は 'UTF-8' に変更
};
```

5. **保存** (Ctrl+S) する

### STEP 3: スプレッドシートを再読み込み
1. スプレッドシートの画面に戻る
2. ブラウザの **更新ボタン（F5）** を押す
3. メニューバーに **「📊 売上CSV取込」** が追加される

### STEP 4: 初回実行
1. メニュー **「📊 売上CSV取込 → CSVを取り込む」** をクリック
2. 初回は **権限の承認** を求められる → **「許可」** をクリック
3. 確認ダイアログが表示 → **「はい」** で実行
4. 完了メッセージが出ればOK！

---

## 機能一覧

| メニュー項目 | 説明 |
|---|---|
| CSVを取り込む | ドライブから最新CSVを読み込み、シートに転記 |
| フォルダ内のCSV一覧を表示 | 対象フォルダ内のCSVファイルを一覧表示 |
| 設定を確認 | 現在の設定値を表示 |

---

## トラブルシューティング

| 症状 | 対処法 |
|---|---|
| 文字化けする | `CSV_ENCODING` を `'Shift_JIS'` に変更 |
| CSVが見つからない | フォルダIDの確認、ファイル名にパターン文字が含まれているか確認 |
| 権限エラー | GAS初回実行時の権限承認を再度行う |
| メニューが出ない | スプレッドシートを再読み込み（F5） |

---

## 自動実行（トリガー設定）

毎日自動で取り込みたい場合:
1. Apps Scriptエディタ → 左メニュー「⏰ トリガー」
2. 「トリガーを追加」をクリック
3. 実行する関数: `importCSVFromDrive`
4. イベントのソース: `時間主導型`
5. 時間ベースのトリガータイプ: `日付ベース`
6. 時刻: お好みの時間帯を選択

> ⚠️ 自動実行の場合、確認ダイアログは表示されません（UI操作なしで自動転記されます）。
