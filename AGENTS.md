## プロジェクト概要

Streamlit製のシフト自動作成ツール（Python）。
スタッフの勤務条件・役割・休暇希望に基づき、ビームサーチアルゴリズムでシフトを自動生成する。
Flet版への移行も並行開発中（`flet_app/`）。

## 言語・スタイル

- **言語**: Python 3.x
- **コメント**: 日本語
- **スタイル**: PEP 8 準拠

## 重要ファイル

| ファイル | 役割 |
|----------|------|
| `app.py` | Streamlit メインアプリ |
| `flet_app/main.py` | Flet版エントリーポイント |
| `shift_settings.json` | 保存された設定データ（スタッフ情報・休暇・日付範囲） |
| `.serena/memories/` | プロジェクトコンテキスト（Serena MCP） |

## 作業ルール

1. **作業前**: `docs/ai_best_practices/best_practices.md` とSerenaメモリを確認する
2. **再実装禁止**: `key_tasks` の完了済み項目は再実装しない
3. **確認**: 実装前にユーザーへ計画を説明し承認を得る
4. **コメント**: コードコメントは日本語で記述する

## 禁止事項

- `shift_settings.json` の手動削除（ユーザーデータが消失する）
- `applymap` の使用（pandas新版では `map` を使うこと）
- 実装済み機能の再実装（Serena `key_tasks` で確認すること）
- ユーザー確認なしの実装開始
